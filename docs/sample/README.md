# Docs Viewer Demo

Este repo es solo para demostrar el visor in-app de documentos.

## Contenido

- `README.md` — este archivo (Markdown).
- `dummy.pdf` — un PDF pequeño para probar el iframe inline.

## Características probadas

1. Render de Markdown con `react-markdown` + `remark-gfm`.
2. Render de PDF inline vía `<iframe>` + blob URL.
3. Botón "Abrir en GitHub" como acción secundaria.
